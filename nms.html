3D Print - Sentinel No Man's Sky

fascinated by No man's sky, and because I wanted to 3D print something pretty complex, I decided to print a sentinel Walker form this game.

First thing First, I can't do 3D model, I tried but I'm just really bad at this. So the first solution that came to my mind, search in the game directory for models but everything is compiles in big binaries files. the second option was to rip the mesh model from the game extracting the directX instructions, I knew with Intel graphics performance analyzers it is possible.

I realized pretty quickly that it couldn't work, because No Man's Sky use OpenGL library, so this tool will not work. I tried several opengl rip tool but nothing worked.

Then I found this tool on github : NMS-Tools this tool can be used to decompile the Mbin files and view the moodels, but you can export those models to .obj or .stl files. But a good news is that NMS-Tool use the DirectX library to render the models. So by using Intel graphics performance analyzers, I was to able to export the sentinel walker model to a .obj.

After cuting the model in three parts with blender, to print separatly body and legs, et Voila:
