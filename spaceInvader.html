Space Invader

In 2009, while I was studying, I decied to start a personnal project : Space Invader clone write in Java/processing. It worked, I didn't use sprite, the game worked fine, smooth and without crach or bug. But years later I noticed that 3 project reuse my Space Invader -see below-, so I reopened the code .. it was awfull, the code was written with the spagetti pattern, a fake OOP approach, magic numbers. So end of 2014 I decided to recode the sketch, with real OOP approach, justclean this code. It will be easyer to use. Months later I restart same project wrote with Unity / C# => Github repo : Github
