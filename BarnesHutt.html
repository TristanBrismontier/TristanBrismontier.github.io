Barnes-Hut-Algorithm

This simulator uses the Barnes-Hut algorithm to simulate gravity on n bodies. It is notable for having order O(n log n) compared to a direct-sum algorithm which would be O(nÂ²).

Performance

This solution, using Barnes-Hut, perform better than the brute force solution. Despite approximation the behavior remain very similar to the brute force. Execution time for N Body N->[1-1000], 10 mesures per N.
